# ⚡ Advanced Ping Tester Framework v2.0
> **Developer:** Imran (রানা ভাই / RANA VHAI)  
> **Platform:** Termux / Pydroid 3 (Android Environment)  
> **Type:** Network Diagnostics & Latency Tester

একটি অত্যন্ত কার্যকরী এবং দ্রুতগতির নেটওয়ার্ক ডায়াগনস্টিক টুল। এটি যেকোনো আইপি অ্যাড্রেস (IP Address) বা ডোমেন নেমের (Domain) পিং (Ping) টেস্ট করে নেটওয়ার্কের লেটেন্সি (Latency), স্ট্যাবিলিটি এবং কানেকশন স্পিড লাইভ চেক করার জন্য তৈরি করা হয়েছে।

---

## 🚀 বৈশিষ্ট্যসমূহ (Features)
* **লাইভ স্পিড ও লেটেন্সি টেস্ট:** সার্ভার কত দ্রুত রেসপন্স করছে তা মিলি-সেকেন্ডে (ms) লাইভ প্রদর্শন করে।
* **ডোমেন অটো-ডিটেকশন:** আইপি ছাড়া সরাসরি ডোমেন নেম (যেমন: google.com) দিয়েও সহজে পিং করা যায়।
* **প্যাকেট লস অ্যানালাইসিস:** নেটওয়ার্কে কোনো ডেটা লস হচ্ছে কিনা তা নিখুঁতভাবে হিসাব করে।
* **টার্মিনাল ইউজার ইন্টারফেস:** আকর্ষণীয় ASCII আর্ট ব্যানার এবং প্রফেশনাল কালারফুল আউটপুট।

📋 ব্যবহার বিধি (How to Use)
​টুলটি চালু হওয়ার পর আপনার টার্গেট হোস্ট (IP Address বা Domain Name) ইনপুট দিন।
​আপনি কতবার পিং পাঠাতে চান (Ping Count) তা সিলেক্ট করুন।
